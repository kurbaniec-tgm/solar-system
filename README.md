## About
Simple solar system simulation

## 🚴 Usage

### 🛠️ Build and Run

```
wasm-pack build --release --no-typescript
cd www
npm run start
```

### 🔬 Test in Headless Browsers with `wasm-pack test`

```
wasm-pack test --headless --firefox
```
